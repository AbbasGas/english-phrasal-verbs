This is a application to learn phrasal verbs. The app selects a random phrasal verb, displays definition and expects you to pass checks.
If you fail the check, you'll be returned to previous next time. If all tests are complete, the verb will be removed from the list and a new verb will be suggested. List length-option define how much verbs you're learning.

Check list for phrasal verb
  • Combine a answer from 3 verbs and 3 prepositions
  • Select one of 3 definitions by verb
  • Combine a answer from 5 verbs and 5 prepositions
  • Select one of 3 synonyms by verb
  • Select one of 5 definitions by verb
  • Combine a answer from 8 verbs and 8 prepositions
  • Reconstruct the verb in example
  • Select one of 5 synonyms by verb
  • Reconstruct the preposition in example

If app don't work properly, e.g. you don't listen any sound or you see permanent "Loading..." instead "Play"-button on first screen, it's means that your browser is obsolete. Use Chrome to avoid troubles.
Minimal supported Android version is 4.1 with Chrome 69.
